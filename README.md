# Nano SQL
If you want to check this app

## Requirements
* yarn (or npm)

## Development

``` bash
# Clone repo
git clone https://github.com/juanem1/nano-sql.git

cd nano-sql

# install all dependencies
yarn

# build app and open in a new window
npm run start
```
